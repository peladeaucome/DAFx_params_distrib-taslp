Abstract

Recent studies have proposed to synthesize audio signals using explicit digital signal processing algorithms which parameters are estimated using deep learning models. These models are optimized such that the output of the algorithm matches a given target input signal. To achieve this, the algorithm should be differentiable, hence the name differentiable digital signal processing (DDSP). However, these algorithms are usually not injective functions w.r.t. their parameters, i.e. multiple different parameter combinations can generate the same output. This is in contradiction with recent studies which attempt to estimate a single set of parameters in a deterministic way: they output a single parameter combination to synthesize a given sound. In this work, we propose a probabilist model that learns all these combinations. For this, we propose to extend the variational auto-encoder (VAE) model (which relies on a single Gaussian distribution) to the use of mixtures of Gaussians (MoG) followed by normalizing flows (NF). It is used to model the target parameter distribution. We optimize our model using the sum of two loss terms: an audio reconstruction loss computed using differentiable signal processing (so that the parameters accurately reproduce the target sound) and an entropy loss, which enforces the model to output diverse parameter combinations. We evaluate successfully our approach on two tasks: estimating the parameters of a frequency modulation (FM) synthesizer and of a full mastering effects chain (equalizer, dynamic range compressor, clipper).


We estimate the distributions using a normalizing flows framework.
